# Deshan & Janisha Wedding Website

A beautiful wedding invitation website showcasing our special day on June 25th, 2026.

## Features

- Elegant old money theme design
- Fully responsive for mobile and desktop
- Interactive countdown timer
- RSVP form integration
- Wedding schedule and venue information
- Photo gallery with stylish clip-path effects

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```
